### Посты (шаг 3 цикла)

- [2026-05-04] Никогда не использовать оборот «Это не X, это Y» (и его варианты: «X — это не Y, это Z», «Не X. Y»).
  → причина: в пилотной партии 2026-05 этот оборот делал текст «ИИ-шным», шаблонным. Сразу выдаёт сгенерированную природу.
  → вместо: формулировать одной прямой мыслью; либо показать различие через сюжет, не называя его словом «не».

- [2026-05-04] Минимизировать тире (—). Каждое тире должно быть осознанным, не автоматическим.
  → причина: переизбыток тире придаёт тексту рваный, «ИИ-шный» ритм.
  → вместо: запятые, точки, переформулировка в текучие предложения. На пост ≤500 знаков — не более 1–2 тире.

- [2026-05-04] Сюжетные посты со сценами хорошо работают — это один из сильных инструментов, но не обязательное правило для каждого поста.
  → причина: в пилоте 2026-05 посты-истории откликнулись сильнее постов-разборов. Но «история» не равно «обязательно сцена». Короткие наблюдения, монологи и афоризмы тоже работают, если они конкретные и в голосе автора.
  → формы, которые можно чередовать:
    1. сцена через «вы» / «ты» — типовая ситуация, в которую читатель встаёт сам;
    2. сцена через «я» — личная зарисовка из жизни Ксении (кухня, кошки, прогулка, кабинет);
    3. case-vignette с именованным клиентом (Маша, Олег) — уместно в тредах, в funnel-end, в сценах с диалогом;
    4. сцена «он / она» без имён — наблюдение со стороны;
    5. монолог боли в кавычках без рассказчика;
    6. короткое наблюдение / афоризм / прямая мысль без сцены вовсе.
  → чего избегать в любой форме: учительского разбора в стиле «у некоторых людей бывает», списков «10 признаков», абстрактных обобщений без конкретики или голоса автора.
  → проверка по партии: если в большинстве постов появились имена (Маша, Лена, Аня…) — это перебор, разнести по 2–3 формам. Если все посты — длинные сцены, тоже плохо: формат должен дышать.

## Формат записи нового правила

```
- [YYYY-MM-DD] <короткое правило одной фразой>
  → причина: <почему — конкретный случай или принцип>
  → вместо: <как надо делать>  (только если NEVER)
```

Пример:
```
- [2026-05-15] Не использовать слово «прокачать» в постах про самооценку.
  → причина: коучинговая лексика, не моя — в посте 014 звучало чужеродно.
  → вместо: «развить», «укрепить», «дать себе опору».
```
