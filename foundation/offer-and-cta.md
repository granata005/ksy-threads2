# Офферы и CTA

> Два CTA работают параллельно. Распределение по постам — в `04-funnels.md` каждого цикла.

## CTA-1. Онлайн-консультация

**Что:** индивидуальные сессии у Ксении в гештальт-подходе. Онлайн.

**Кому подходит:** широкая русскоязычная аудитория, география не важна. Запросы — все, с которыми работает Ксения (см. `audience-decoded.md`): самооценка, тревога, отношения, апатия, эмоциональная регуляция.

**Куда ведём:** `https://ksy-psy.ru` (или Telegram для записи: `@ksksksksjsjs`).

**Когда вставляем в посте:** в финальном посте каждой воронки (≈1 из 5–7), когда тема поста — про сложность, с которой человек уже устал биться в одиночку.

**Формулировки (брать поочерёдно, не повторять подряд):**

1. «Если вам сейчас не хватает опор — приходите на консультацию: ksy-psy.ru».
2. «Если хочется разобраться, что под этой [тревогой / апатией / самокритикой] на самом деле — это можно сделать в терапии: ksy-psy.ru».
3. «А если этим хочется заняться не в одиночку — записаться ко мне на сессию можно через сайт ksy-psy.ru».
4. «Если вы узнали себя и устали быть с этим один на один — приходите. Запись на сайте: ksy-psy.ru».
5. «Это та работа, которую сложно делать самому. Если хочется попробовать вместе — ksy-psy.ru».

**Чего избегать в этом CTA:**
- срочности («запишитесь сейчас, мест мало»),
- скидок и ограниченных предложений,
- обещаний результата («за 10 сессий вы…»),
- жёсткого «вы должны».

## CTA-2. «Кофе с психологом» в Батуми

**Что:** живые тематические встречи, 5–6 человек в группе. Теоретическая часть + обсуждение, как тема применяется в жизни участников. Темы: эмоции, самооценка, отношения, дружба, конфликты.

**Кому подходит:** аудитория в Батуми и окрестностях (грузинская русскоязычная, релоканты, путешественники). География — критична.

**Куда ведём:** Telegram-канал с анонсами `https://t.me/+7me4cd0e10gwMjI6`. (Точная ссылка — из её существующих постов; при первом цикле проверить актуальность.)

**Когда вставляем в посте:** когда тема поста — про общение, отношения, групповой опыт, одиночество в смысле «нет среды единомышленников». Хорошо ложится на посты, где есть мысль «с этим помогает Другой» — встреча и есть Другой.

**Формулировки:**

1. «А если вы в Батуми — приглашаю на ближайшую встречу "Кофе с психологом". Там как раз обсуждаем такие темы вживую. Анонсы здесь: t.me/+7me4cd0e10gwMjI6».
2. «Если читаете это из Батуми — у меня есть формат живых встреч на 5–6 человек по подобным темам. Анонсы в Telegram: …».
3. «Кстати, в Батуми у меня регулярно проходят встречи "Кофе с психологом" — небольшая группа, разбираем такие штуки вместе. Расписание: …».
4. «Для тех, кто в Батуми — ближайшее "Кофе с психологом" будет про [X]. Записаться можно в канале: …».

**Чего избегать:**
- упоминания цены прямо в посте Threads (это в канале),
- двойного оффера в одном посте (либо CTA-1, либо CTA-2 — не оба сразу).

## Когда какой CTA в воронке

Простое правило для воронок (`04-funnels.md`):

| Тип цепочки (по теме) | Финальный CTA |
|---|---|
| Самокритика / самооценка / «со мной что-то не так» | CTA-1 (терапия) |
| Тревога / накручивание / контроль | CTA-1 (терапия) |
| Апатия / пустота / «не своя жизнь» | CTA-1 (терапия) |
| Близость / отношения / страх отвержения | CTA-1 (терапия), иногда CTA-2 в Батуми |
| Эмоциональная регуляция / «слишком чувствительная» | CTA-1 (терапия) |
| Одиночество / «нет среды» / «не с кем поговорить» | CTA-2 (Батуми) с подсказкой про CTA-1 для не-местных |
| Групповой опыт / «как это у других» / «нормально ли я реагирую» | CTA-2 (Батуми) |

В одном цикле — 4 воронки. Распределение по умолчанию: **3 финала на CTA-1, 1 финал на CTA-2** (т.к. широкая аудитория > локальная). Если для конкретного месяца есть ясный анонс встречи в Батуми — поднимаем долю CTA-2.

## Универсальный софт-CTA в постах БЕЗ оффера

Если пост не финал воронки — никакого продающего CTA. Финал поста: либо вопрос к читателю («А у вас как?»), либо открытая фраза («Это то, что точно стоит замечать»). Это калибровка по голосу автора (см. `style-guide.md` → CTA-паттерны).
